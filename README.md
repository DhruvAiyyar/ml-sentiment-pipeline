# ML Sentiment Analysis Pipeline

Machine learning project that trains a sentiment classifier and deploys it via Flask API.

## Installation

1. Create virtual environment: `python -m venv venv`
2. Activate: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
3. Install: `pip install -r requirements.txt`

## Running

- Train model: `python train_model.py`
- Start API: `python app.py`
- Run tests: `pytest tests/ -v`
